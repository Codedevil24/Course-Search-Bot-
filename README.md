# Code Devil Course Bot v2.2

## Features
- Direct bot search
- /search command
- Normal text search
- Categories
- Featured courses
- Typo suggestions
- Inline search
- Paid course selling flow
- Premium access grant
- CSV import
- Telegram photo thumbnail save
- Active users analytics
- Render web service compatible

## Run
pip install -r requirements.txt
Create .env
python app.py
