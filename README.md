# Code Devil Course Bot v2.3

## Features
- Supabase/Postgres database
- Direct bot search
- /search command
- Normal text search
- Categories
- Featured courses
- Typo suggestions
- Inline search
- Paid course selling flow
- Premium access grant
- CSV import
- Telegram photo thumbnail save
- Active users analytics
- Render web service compatible

## Install
pip install -r requirements.txt

## Setup
1. Copy .env.example to .env
2. Fill BOT_TOKEN, ADMIN_IDS, SUPABASE_DB_URL
3. Run:
python app.py
